## Studi Kasus Tingkat Pemula

### 1. Kalkulator Sederhana

Buat sebuah program kalkulator sederhana yang dapat melakukan operasi penjumlahan, pengurangan, perkalian, dan pembagian. Program menerima dua angka dan operator sebagai input, lalu menampilkan hasilnya.

### 2. Konversi Satuan Panjang

Buat sebuah program yang dapat mengkonversi satuan panjang dari meter ke centimeter, kilometer, inci, dan kaki. Program menerima input panjang dalam meter, lalu menampilkan hasil konversi ke satuan-satuan lainnya.

### 3. Program Tebak Angka

Buat sebuah program permainan tebak angka. Program memilih angka acak antara 1 dan 100, lalu pemain harus menebak angka tersebut. Program memberikan petunjuk "terlalu tinggi" atau "terlalu rendah" sampai pemain berhasil menebak angka yang benar.

## Studi Kasus Tingkat Menengah

### 4. Manajemen Kontak

Buat sebuah program manajemen kontak yang dapat menyimpan nama, nomor telepon, dan alamat email dari beberapa kontak. Program menyediakan fitur untuk menambah, menghapus, mencari, dan menampilkan daftar kontak.

### 5. Analisis Teks Sederhana

Buat sebuah program yang dapat menerima input teks, lalu melakukan analisis sederhana seperti menghitung jumlah kata, jumlah karakter, dan mencari kata yang paling sering muncul.

### 6. Permainan Batu Gunting Kertas

Buat sebuah program permainan batu, gunting, kertas melawan komputer. Program menerima input pilihan pemain (batu, gunting, atau kertas), lalu komputer memilih secara acak. Program menentukan pemenang berdasarkan aturan permainan.

## Studi Kasus Tingkat Lanjutan

### 7. Sistem Manajemen Perpustakaan

Buat sebuah sistem manajemen perpustakaan yang dapat menyimpan informasi buku (judul, penulis, ISBN, status ketersediaan) dan anggota perpustakaan (nama, ID anggota). Sistem menyediakan fitur untuk menambah buku baru, meminjam buku, mengembalikan buku, mencari buku, dan menampilkan daftar buku yang tersedia.

### 8. Analisis Data Cuaca

Buat sebuah program yang dapat membaca data cuaca dari sebuah file (misalnya, file CSV) dan melakukan analisis seperti menghitung suhu rata-rata, suhu tertinggi, suhu terendah, dan menampilkan grafik perubahan suhu harian.

### 9. Aplikasi Chat Sederhana

Buat sebuah aplikasi chat sederhana yang memungkinkan beberapa pengguna untuk saling mengirim pesan. Aplikasi dapat menggunakan socket untuk berkomunikasi antara client dan server.
